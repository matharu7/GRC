# 🏢 3. How Companies Apply Frameworks (Real Scenario)

---

## 🔸 Example: ISO 27001 Implementation

```text id="grc205"
Steps:
1. Define scope (which systems)
2. Identify assets (servers, data, apps)
3. Perform risk assessment
4. Apply security controls
5. Document policies
6. Conduct audit
7. Get certification
```

Reference:

* International Organization for Standardization ISO 27001

---

## 🔸 Example: GDPR Implementation

```text id="grc206"
Steps:
1. Identify personal data
2. Get user consent
3. Encrypt sensitive data
4. Define data retention policy
5. Setup breach reporting process
```

Reference:

* General Data Protection Regulation

---

## 🔸 Example: PCI DSS Implementation

```text id="grc207"
Steps:
1. Secure payment systems
2. Encrypt card data
3. Restrict access
4. Monitor logs
5. Perform regular scans
```

Reference:

* Payment Card Industry Data Security Standard

---

## 🔸 Example: NIST Framework Implementation

```text id="grc208"
Steps:
1. Identify assets & risks
2. Protect systems
3. Detect threats
4. Respond to incidents
5. Recover systems
```

Reference:

* National Institute of Standards and Technology

# 🧠 4. Real Company Scenario (Important for Teaching)

## 📌 Scenario 1: Data Breach

```text id="grc209"
Situation:
Company database leaked

GRC Action:
- Identify affected data
- Check compliance (GDPR)
- Report breach within 72 hours
- Update security controls
```

---

## 📌 Scenario 2: Weak Password Policy

```text id="grc210"
Situation:
Employees using weak passwords

GRC Action:
- Update password policy
- Enforce strong password rules
- Conduct awareness training
```

---

## 📌 Scenario 3: Third-Party Risk

```text id="grc211"
Situation:
Using insecure vendor service

GRC Action:
- Perform vendor risk assessment
- Ensure compliance (ISO/NIST)
- Replace or secure vendor
```

# 📄 5. Sample Policy (Basic Example)

```text id="grc212"
Policy Name: Password Policy

Rules:
- Minimum 8–12 characters
- Must include uppercase, lowercase, number, special character
- Change every 90 days
- No reuse of last 5 passwords
```
